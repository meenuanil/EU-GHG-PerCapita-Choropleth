## üìå Project Description
This repository presents an **academic choropleth map** visualizing **Greenhouse Gas (GHG) emissions per capita across European Union countries**.  
Developed as part of an **Advanced Cartography course**, the project emphasizes **spatial accuracy**, **cartographic design**, and **thematic clarity** for climate data communication.

---

## üéØ Objectives
- Visualize spatial distribution of GHG emissions per capita in the EU  
- Apply advanced thematic cartography techniques  
- Ensure area-preserving projection for continental analysis  
- Demonstrate GIS-based data integration workflows
